# Java-Gym-app
Gym application

This application is created by using NetBeansJDE. The application is created during studying at school for studying purpose. It contains most of the basic ideas about java including exception handling, thread. This application doesn't have any database, but it contains an array to do that function of database. 

How to use?
The user need to sign up to get an account, then they choose the plan they want (14 days or 28 days). it get the BMI of the first day and the last day of the plan. Then comparison between them. So the user can see the different.
The app also contains some excersise videos from youtube with reliable source.
It has a nutrition supplier (BodyBuilding.com) one of the famous in the world. Reliable and trustable.
User can update their information as well.
